## üì± How to Connect (3 Easy Ways)

### Option 1: Email Login (Easiest!)

1. Click the **"Connect Wallet"** button
2. Select **"Email"**
3. Enter your email address
4. Check your email for a 6-digit code
5. Enter the code
6. Wait 10-30 seconds
7. ‚úÖ You're connected!

**Tip:** Check spam folder if code doesn't arrive in 2 minutes.

### Option 2: Google Login (Fast!)

1. Click the **"Connect Wallet"** button
2. Select **"Google"**
3. Choose your Google account
4. Authorize the connection
5. Wait 10-30 seconds
6. ‚úÖ You're connected!

**Tip:** Allow popups in your browser for this to work.

### Option 3: Web3 Wallet (For Crypto Users)

1. Install MetaMask, Phantom, or any wallet
2. Click the **"Connect Wallet"** button
3. Select **"Wallet"**
4. Choose your wallet
5. Approve the connection
6. Wait 10-30 seconds
7. ‚úÖ You're connected!

**Tip:** Works with wallets from ANY chain (Ethereum, Solana, etc.)
